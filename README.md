## Important Instructions

To ensure a fair and effective learning experience, please adhere to the following guidelines:

1.  **Disable AI Code Completion Tools**: When working on these assignments, please disable any AI code completion tools (e.g., GitHub Copilot, Codeium, Tabnine). The goal is to solve the problems using your own understanding and knowledge.

2.  **Avoid Checking Test Logs for Answers**: When running tests, focus on the test results (pass/fail) to identify issues. Do not inspect the test logs or output to directly find the answers or correct implementations, as this defeats the purpose of the exercises.

Happy Coding! 💛
